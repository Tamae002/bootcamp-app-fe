# Geeksfarm Bootcamp App Frontend
---------------------------------

# System Requirement
- **OS**: Windows, Linux, atau MacOS
- **Node.js:** versi 18.x atau lebih tinggi

# Quickstart

## Instalasi Dependensi
Sebelum mulai menjalankan server/melakukan pengembangan, pastikan semua dependensi sudah diinstal. Jalankan perintah berikut untuk melakukan instalasi dependensi:
```sh
npm install
```

## Cara menjalankan
### Lingkungan Pengembangan (Development)
1. Jalankan perintah berikut di terminal/command line (cmd) untuk menjalankan server pengembangan:  
```sh
npm run dev
```
atau
```sh
vite
```
Secara default, server akan berjalan di `http://localhost:5173` atau sesuai dengan output/instruksi pada terminal.

2. Buka browser dan pergi ke url `http://localhost:5173` atau sesuai output pada terminal.
